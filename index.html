1. Problem Statement

Project Goal: Implement a DMP-based algorithm and neural network to solve the Robosuite Nut Assembly Task using expert demonstrations. Compare the training time and accuracy of each of the methods.

We planned to implement two separate methods for learning a policy from data:
  1. A DMP-based algorithm, similar to a higher-dimensional version of the program implemented in Coding Assignment 3
  2. A neural network capable of directly mapping sensor inputs to control outputs
We hope to train both programs on the provided data and compare their performance.

The Robosuite Nut Assembly Task

Success Criteria

Criteria for a successful policy: Trained policy can successfully & consistently complete the Square Assembly task, including for targets with varying orientations.

Goal: Develop successful policies (i.e. meeting the above criteria) using separate DMP and deep learning approaches, and compare the results.
  • Metrics for comparison: success rate and average time to complete the task
